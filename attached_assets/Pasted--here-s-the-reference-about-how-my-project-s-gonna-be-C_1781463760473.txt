## What we're building

**CarbonEra** is a full-stack, scroll-driven, AI-powered website that makes carbon footprint awareness deeply personal and emotionally resonant. It's not a dashboard. It's an experience вЂ” a journey from 1750 (industrial revolution) to the present day, and then into the user's own life.

The user lands, scrolls through history, enters their lifestyle details, gets an AI-generated personal carbon story, sees their gamified footprint breakdown, and leaves with a shareable Carbon Card.

**Target:** Google PromptWars challenge вЂ” top 25 out of 900+ students.

---

## Core user journey (in order)

1. **Landing / History scroll** вЂ” animated Earth globe, CO2 rising from 1750 в†’ 2024 with real OWID data, GSAP scroll triggers
2. **Present day crisis** вЂ” live CO2 PPM counter, live India grid emissions via CO2Signal API
3. **Personal quiz** вЂ” 60-second lifestyle quiz (city, diet, transport, energy, flights)
4. **Footprint result** вЂ” your personal CO2 number, compared to India avg / global avg / Paris target
5. **Gamified breakdown** вЂ” 4 category XP bars (diet, transport, energy, shopping), level labels, unlock mechanic
6. **AI story** вЂ” LLM generates a 3-paragraph personalized climate story
7. **Pledge + Carbon Card** вЂ” user picks a habit change, downloads/shares a styled card

## Tech stack (all free)

### Frontend
- **React + Vite** вЂ” fast dev, HMR, easy deployment
- **Tailwind CSS** вЂ” utility-first, pairs with component libraries
- **GSAP + ScrollTrigger** вЂ” scroll animations, timeline reveals, number counters
- **Three.js** вЂ” Earth globe on the landing section (animated CO2 haze builds up as user scrolls)
- **Framer Motion** вЂ” micro-interactions, page transitions, quiz step animations
- **Recharts** вЂ” CO2 historical chart (1750вЂ“2024), trajectory vs Paris target

### UI Component libraries
- **shadcn/ui** вЂ” for quiz inputs, buttons, progress bars, cards. Install: `npx shadcn@latest init`
- **Aceternity UI** вЂ” for the hero background effects (aurora, spotlight, particle effects). Copy components directly from https://ui.aceternity.com/components
- **Lucide React** вЂ” icons (already included with shadcn)

### Backend
- **Node.js + Express** вЂ” REST API server
- **MongoDB Atlas** (free tier, 512MB) вЂ” stores quiz submissions, city leaderboard, pledge counts
- **Mongoose** вЂ” ODM for MongoDB
- **Redis via Upstash** (free tier) вЂ” rate limiting on AI endpoint (max 1 LLM call per session), caching live API responses
- **Winston** вЂ” structured logging
- **cors, dotenv, express-rate-limit** вЂ” standard middleware

### Hosting (all free)

## How to call the LLM API

### Using Claude (Anthropic) вЂ” recommended

```js
// server/services/llmService.js
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

export async function generateCarbonStory(userData) {
  const { city, totalCO2, dietScore, transportScore, energyScore, topCategory } = userData;

  const prompt = `
You are a climate scientist writing a personal carbon story for someone.

User profile:
- City: ${city}
- Annual carbon footprint: ${totalCO2} kg CO2 (India avg: 1900 kg, global avg: 4800 kg)
- Biggest impact area: ${topCategory}
- Diet score: ${dietScore}/100, Transport score: ${transportScore}/100, Energy score: ${energyScore}/100

Write exactly 3 short paragraphs:
1. What their footprint means for ${city} in 2050 if everyone lived like them (visceral, specific)
2. The one change that would save the most CO2 for their profile (with exact kg savings)
3. One surprising fact about their biggest impact area (${topCategory}) that most people don't know

Keep it personal, honest, and non-preachy. Under 200 words total.
  `;

  const message = await client.messages.create({
    model: "claude-opus-4-6",
    max_tokens: 400,
    messages: [{ role: "user", content: prompt }],
  });

  return message.content[0].text;
}
```

## How to use GSAP ScrollTrigger

```bash
npm install gsap
```

```jsx
// client/src/components/HistoryScroll.jsx
import { useEffect, useRef } from "react";
import gsap from "gsap";
import { ScrollTrigger } from "gsap/ScrollTrigger";

gsap.registerPlugin(ScrollTrigger);

export function HistoryScroll() {
  const containerRef = useRef();

  useEffect(() => {
    const ctx = gsap.context(() => {
      // Animate year counter from 1750 to 2024 as user scrolls
      gsap.to(".year-display", {
        innerText: 2024,
        snap: { innerText: 1 },
        scrollTrigger: {
          trigger: containerRef.current,
          start: "top top",
          end: "bottom bottom",
          scrub: 1,
        },
      });

      // Fade in each timeline event
      gsap.utils.toArray(".timeline-event").forEach((el) => {
        gsap.from(el, {
          opacity: 0,
          y: 40,
          scrollTrigger: { trigger: el, start: "top 80%", toggleActions: "play none none reverse" },
        });
      });
    }, containerRef);

    return () => ctx.revert();
  }, []);

  return (
    <section ref={containerRef} className="min-h-[500vh] relative">
      <div className="sticky top-0 h-screen flex items-center justify-center">
        <span className="year-display text-8xl font-bold text-white">1750</span>
      </div>
      {/* Timeline events absolutely positioned along scroll height */}
    </section>
  );
}
```

## Shareable Carbon Card (html-to-image)

```bash
npm install html-to-image
```

```jsx
import { toPng } from "html-to-image";
import { useRef } from "react";

export function CarbonCard({ totalCO2, city, pledge, level }) {
  const cardRef = useRef();

  const download = async () => {
    const dataUrl = await toPng(cardRef.current);
    const link = document.createElement("a");
    link.download = "my-carbon-card.png";
    link.href = dataUrl;
    link.click();
  };

  return (
    <>
      <div ref={cardRef} className="w-80 p-6 bg-gradient-to-br from-emerald-900 to-slate-900 rounded-2xl text-white">
        <p className="text-xs uppercase tracking-widest text-emerald-400 mb-1">CarbonEra</p>
        <h2 className="text-4xl font-bold">{(totalCO2 / 1000).toFixed(1)}t COв‚‚</h2>
        <p className="text-sm text-slate-300 mt-1">{city} В· {new Date().getFullYear()}</p>
        <div className="mt-4 pt-4 border-t border-white/10">
          <p className="text-xs text-slate-400">My pledge</p>
          <p className="text-sm font-medium mt-1">{pledge}</p>
        </div>
        <div className="mt-3 inline-block bg-emerald-500/20 text-emerald-300 text-xs px-3 py-1 rounded-full">
          {level}
        </div>
      </div>
      <button onClick={download} className="mt-4 w-full py-2 border border-white/20 rounded-lg text-sm">
        Download card
      </button>
    </>
  );
}
```
