# Carbon Footprint Awareness

Premium vanilla landing page hero for a carbon footprint awareness platform.

## Structure

```text
Carbon_Footprint_Awareness/
  index.html
  assets/
    images/
      hero-pollution.png
      hero-nature.png
  src/
    scripts/
      main.js
    styles/
      main.css
```

## Run

Open `index.html` in a browser, or serve the folder with any static server.

The current build focuses only on the hero section and cursor-based image reveal. The folders are split so future landing sections, game modules, calculators, dashboards, auth, API integrations, and backend-connected features can be added without rewriting the hero.
